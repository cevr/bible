# Bohr Reading Audit Tracker

For each verse in `bohr-vs-millers-rules.md`, verify Bohr's reading row is grounded in actual Bohr text (with page-cite quotes), and label anything beyond Bohr as Extension/Note. Mark `[x]` when audited.

## Method per verse

1. `grep -n` the relevant Bohr txt file at `bohr/` for the verse reference
2. Read the current Bohr reading row in `bohr-vs-millers-rules.md`
3. Verify quoted material matches Bohr verbatim; rewrite citations as `(Bohr, *<Volume short>*, p. X)`
4. Label any inference beyond Bohr as "Extension beyond Bohr (not in source)" or "Note (not in Bohr)"
5. Mark `[x]` below
