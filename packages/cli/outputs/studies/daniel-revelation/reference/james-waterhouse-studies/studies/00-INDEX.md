## How to Use These Files

Each study file is a **reference sheet** — it contains:

1. **Title and theme** of the study
2. **Chiastic structure** (the literary architecture)
3. **Key texts** with verse references
4. **Cross-references** to other biblical passages
5. **Novel findings** unique to Waterhouse's analysis
6. **Doctrinal connections** to Adventist theology
7. **Teaching flow** — the logical sequence for presenting the material

An agent can use any single file to reconstruct a full Bible study presentation, sermon outline, or teaching session. The cross-references between files show how the studies build on each other.
